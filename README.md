# ZhiHuDaily-知乎日报
  实现功能
-------
  * 启动界面图像获取，动态加载logo <br> 
  * 最新消息及过往消息查看 <br>
  * 下拉刷新，自动加载更多内容<br>
  * 新闻对应长短评论查看 <br>
  * 主题日报列表及内容查看 <br>
  * 自动离线下载浏览过的新闻列表，内容和收藏的文章<br>
  * 夜间模式 <br>
  * 一键返回顶部 <br>
  * 查询指定日期新闻 <br>
  * 多账号登录登出 <br>
  * 收藏及取消收藏相应文章 <br>
  * 以及其他设计的小细节<br>
